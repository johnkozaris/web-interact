# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

**Email:** ioanniskozaris@gmail.com

Do not open a public issue for security vulnerabilities.

I will respond within 48 hours and work on a fix. Once resolved, I will credit you in the release notes (unless you prefer anonymity).

## Scope

This policy covers the `web-interact` CLI binary, the Node.js daemon, and the QuickJS sandbox. It does not cover Chrome/Chromium itself or Patchright upstream.
